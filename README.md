# Windows 12 Web OS Clone

A modern and highly interactive Windows 12-inspired Web Operating System built using FastAPI, HTML, CSS, and Vanilla JavaScript.

This project simulates a real desktop operating system experience directly inside the browser, including:

* Boot screen
* Lock screen
* Login & authentication system
* Desktop environment
* File Explorer
* Start Menu
* Taskbar
* Widgets panel
* Notifications
* Media players
* File associations
* Dynamic app windows
* Virtual filesystem
* Google Sheets cloud storage integration

The system is designed to behave like a real Windows 11/12 operating system with professional UI/UX and dynamic file handling.

## Features

* Realistic Windows-style desktop interface
* Dynamic window management system
* File Explorer with folder navigation
* Professional file association system
* Image, audio, video, PDF, and code file support
* FastAPI backend with JWT authentication
* Google Sheets used as cloud database/storage
* Media rendering instead of raw Base64 text
* Context menus and desktop interactions
* Responsive modern UI
* Modular backend architecture
* Custom boot animation and lock screen
* Virtual apps system

## Current Goal

The goal of this project is to create a browser-based next-generation Web OS that feels as close as possible to a real desktop operating system.

Future updates will include:

* Terminal emulator

* Multi-user sessions

* Advanced media applications
* Virtual memory management
